# 🎮 Web Game Platform – Authentication System



## 📌 Overview

This project is a web application developed for the "Web Application Development" course.



It represents the structural foundation of an online game platform, including:



- User registration

- User login

- Session management

- Redirect to game home page

- Database integration



⚠️ The actual game logic is not implemented yet.  

The focus of this project is backend authentication and database interaction.



---



## 🚀 Features



- Secure user registration

- Login authentication

- Session handling

- MySQL database integration

- Input validation

- Basic frontend structure



---



## 🛠️ Technologies Used



- PHP

- HTML

- CSS

- JavaScript

- MySQL



---



## 🗄️ Database Structure



The application uses a relational database to store:



- User credentials

- User information



---



## ▶️ How to Run Locally



1. Install XAMPP or similar local server

2. Import the SQL file into phpMyAdmin

3. Place the project inside the `htdocs` folder

4. Start Apache and MySQL

5. Open `localhost/project-name`
